<analysis>
The AI engineer systematically built the Job Rocket application from an initial UI to a comprehensive platform. The progression began with establishing a tech-rocket themed UI and then implementing a gamified onboarding and authentication system for both job seekers and recruiters, including robust backend models and frontend progress indicators. Subsequent development focused on core functionalities: first, a detailed company structure and team management feature for recruiters (deferring email integration for later), then a comprehensive job listing and posting system for recruiters (including single and bulk uploads, auto-expiry, archiving, and reposting). The latest major addition was the Easy Apply job application system, pre-populating seeker data and enabling recruiter application management. Currently, the engineer is integrating a sophisticated pricing and payment system, using Payfast, with automatic package activation, demonstrating a pragmatic and iterative approach to feature delivery, prioritizing core functionality and addressing dependencies as needed.

<product_requirements>
The Job Rocket application aims to be a comprehensive platform connecting job seekers, recruiters, and administrators, featuring an Applicant Tracking System (ATS), subscription models for job listings and CV search, and payment processing via Payfast.

**Implemented Features and Current State:**
1.  **Job Listing UI & Theme:** An initial modern UI evolved into a techie, rocket feel theme with dark greys, blues, gradients, glassmorphism, and animations.
2.  **Gamified Onboarding (Job Seeker):** Full registration/login. Gamified profile completion (100 points: Profile Picture, About Me, Work History, Skills, Education, Achievements, Intro Video, Apply to 5 Jobs, Email Job Alert) with a visual trophy progress and rocket launch animation at 100 points.
3.  **Gamified Onboarding (Recruiter):** Role selection at registration. Gamified company profile completion (Company Logo, Cover Image, About Company, Number of Employees, Website/LinkedIn links).
4.  **Company Structure & Team Management:** Recruiters can add a main company, branches (optional location/email), and invite team members, associating them with branches. The invitation system stores invites but defers email sending.
5.  **Job Listing & Posting:** Recruiters can post jobs (single or bulk CSV/Excel) with mandatory fields (Title, Location, Salary, Type, Work Type, Industry, Description) and optional (Experience, Qualifications, Company selection). Jobs automatically expire in 35 days, are hidden from seekers but archived for recruiters, and can be reposted. Display shows days/weeks since posted.
6.  **Job Application System:** Optional external application link. If blank, Easy Apply allows job seekers to apply directly on Job Rocket via a modal pre-populated with their profile (name, surname, email, CV, skills, optional cover letter). Job seekers can track applications, and recruiters can view/manage them.
7.  **Pricing & Payment Packages:** Defined packages: Two Listings (R2800 once-off, 30-day expiry), Unlimited Listings (R3899/month, 35-day expiry, 10 free CV searches), Five Listings (R4150 once-off). CV Search packages: 10 CVs" (R699 once-off), "20 CVs (R1299 once-off), Unlimited Searches (R2899/month). CV search functionality is pending, but packages are on the pricing page.
</product_requirements>

<key_technical_concepts>
-   **Full-stack Architecture**: React (frontend), FastAPI (backend), MongoDB (database).
-   **UI/UX**: Tailwind CSS, Shadcn UI components, Glassmorphism, CSS animations.
-   **Authentication**: JWT for role-based authentication.
-   **Database**: PyMongo/Motor for MongoDB, UUIDs for IDs.
-   **Payment Gateway**: Payfast integration (in progress).
-   **Email Integration**: SendGrid (planned, credentials pending).
-   **Data Processing**: Pandas for CSV/Excel bulk uploads.
</key_technical_concepts>
