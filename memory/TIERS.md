## Business Rules

1. **Account Creation**: Auto-create when recruiter signs up (they become owner)
2. **User Limits**: Enforce based on `tier.included_users + account.extra_users_count`
3. **Feature Access**: Check `tier.features` + `account_addons` for gating
4. **Billing**: Monthly subscription, prorated for mid-cycle changes
5. **Downgrade**: Block if current users exceed new tier's limit
6. **Enterprise**: Custom pricing (R39,999+), contact sales flow
